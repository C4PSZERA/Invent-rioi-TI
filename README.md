📱 App Inventário TI - Projeto de Extensão
Aplicativo desenvolvido em React Native para otimização do controle de hardware nas unidades CDD da Ambev no Paraná.

Funcionalidades:
* Lista de notebooks utilizadosFlatList.
* Interação moderna com o componentePressable.
* Focado em arquitetura Offline First para uso em galpões com sinal assustador
